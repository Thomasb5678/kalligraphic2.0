# Images Blog

Placez ici vos images d'articles de blog.

## Nommage recommandé :

- `article1.jpg` - Image pour l'article 1
- `article2.jpg` - Image pour l'article 2
- `nouveau-article-mai-2024.jpg` - Image descriptive

## Format recommandé :

- Taille : 600x400px
- Format : JPG ou PNG
- Poids : < 500KB
